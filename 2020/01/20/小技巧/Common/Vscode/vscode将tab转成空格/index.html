    vscode将tab转成空格

    vscode
    创建时间:2020-01-20 16:03
    阅读:

    按快捷键 Command + Shift + P即（workbench.action.showCommands）查找命令，然后输入 Indent Using Space 回车，会提示将一个 tab 转成几个空格，选择一个自己想要的即可。
